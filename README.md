# Upc-Scanner
    Upc Scanner in react native. Included UPC API.
    Scans upc and returns true if it exists at given endpoint
    Tested on android 7.0.
    
# Mobile
- npm install
- adb reverse tcp:8081 tcp:8081
- react-native run-android
